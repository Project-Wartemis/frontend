What

This is a project to allow bots to play against each other on a game.
Make your own bot and try to win as much as you can!

Why

We often discussed each implementing a bot on certain games, side by side,
investigating which would be better and then to continuously improve upon them to try to beat the other.
After a year we had the idea to just make a general platform on which we could easily implement
any game and any bot we wanted, and open it up to the public.

How

If you want to make your own bot, either start with one of our premade packages
which you can find to the left under Quick Start, or implement
all necessary message parsing logic in the language of your choice. More info here.

After that, decide which game you want to play
(unless you're making a general AI... in that case, ignore this step. Good luck!)
You can see a description of each game on their respective pages to the left.

Before a game starts, you first need to run your bot (on your machine, or wherever you host it).
After that you can create a game in the lobby,
add your bot and any other bots
(You can even add the same bot multiple times!), and start the game.

Your bot will be sent a game state, and your bot should reply with actions.
The format of both of these is completely dependent on what game you're playing.
The underlying technology used between the bot and the server is websockets,
which provides realtime two-way communication.

Tribute

This project was inspired by MOZAIC which held a one-time competition in 2018,
which itself was inspired on the 2010 Fall competition organized by Google and the University of Waterloo.
