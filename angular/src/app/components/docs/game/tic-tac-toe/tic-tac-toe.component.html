1. Mechanics

The game is played on a 3x3 grid. There are 2 players, X and O.
Each player gets to place their symbol on a free position on the 3x3 grid.
Player X starts. The goal is to get 3 in a row (in any row/column or diagonal).
The game is drawn if the board is full and neither player has 3 in a row.
